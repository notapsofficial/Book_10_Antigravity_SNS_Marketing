# 第6章：最初に立ちはだかったAPIという現実

## 1. 無料の時代の終わり

Antigravityに導かれるまま、私は「X Developer Platform」のページを開きました。
そこで私を待っていたのは、厳しい現実でした。

「Free Plan: Write Only (Limit 1,500 tweets/month) ... **but NO API Access for Bots in the way you think**」

かつて、TwitterのAPIは誰でも無料で使い放題でした。
しかし、2026年の今、状況は一変しています。
まともにAPIを使って自動投稿システム（Bot）を作るには、**「Basic Tier」以上（月額100ドル）** の課金が必要だったのです。

「えっ、有料なの？」
私は思わず画面の前で声を上げました。
個人開発者にとって、まだ収益化もしていないアプリの広報のために、毎月100ドル（約1万5千円）を払うのは、決して小さな判断ではありません。

## 2. 他のSNSはどうなのか？

私はAntigravityに頼んで、他の主要SNSのAPI事情も調査させました。

*   **LinkedIn**: APIはあるが、審査が厳しい。個人利用のハードルは高い。
*   **Instagram / Facebook**: Graph API経由で可能だが、ビジネスアカウント化などの手続きが複雑。
*   **Threads**: APIは公開されたばかりで、仕様が頻繁に変わる。
*   **Bluesky**: AT Protocolで開放的だが、まだユーザー数が（Xに比べれば）少ない。

どれも一長一短です。
「無料で、簡単に、多くの人に届く」という虫のいい話は、どこにも落ちていませんでした。

## 3. これは「コスト」か「投資」か

ここで私は立ち止まりました。
「たかがSNSのために、お金と手間をかける価値があるのか？」

しかし、第1章から第3章までの自分の姿を思い出してみました。
SNSをやろうとして挫折し、時間を溶かし、本業の開発が止まったあの日々。
あの「失われた時間」と「精神的ストレス」を時給換算したら、月額100ドルなんて安いものではないか？

企業が広告を出せば、もっと莫大な費用がかかります。
私が作ろうとしているのは、私専用の「全自動広報マシン」です。
そのサーバ代やライセンス料だと思えば、これは**「コスト（浪費）」ではなく「インフラへの投資」**です。

## 4. 覚悟を決める

「Antigravity、Xで行こう」

私は決めました。
一番ユーザーが多く、拡散力があり、そして（有料とはいえ）APIの仕様がドキュメント化されているXを選ぶ。
それは、私がこのプロジェクトに対して「お金を払ってでも解決する」という覚悟を決めた瞬間でもありました。

「分かりました。では、X Developer Portalでプロジェクトを作成し、APIキーを取得しましょう」

Antigravityは事務的に答えましたが、私にはそれが「よく決断した」と言ってくれているように聞こえました。
さあ、ここからが本当の「実装」の始まりです。
クレジットカードを取り出し、私は Developer Portal の "Sign Up" ボタンをクリックしました。

## 到達状態 (Check Point)

*   [ ] 現在のSNS APIが（多くの場合）有料または複雑であることを知った。
*   [ ] 「無料で楽に」という甘えを捨て、「投資」として捉え直した。
*   [ ] 最初のプラットフォームとしてXを選び、Developer Portalに進む決意をした。

覚悟は決まりました。
しかし、この Developer Portal という場所は、慣れていない人間にとっては「迷宮」のような場所でした。
