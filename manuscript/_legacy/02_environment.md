## 1. AIに「場所」を作らせる

まずはプロジェクト用のフォルダと、Pythonの実行環境を用意します。
しかし、黒い画面（ターミナル）に不慣れなコマンドを打ち込む必要はありません。Antigravity（AI）にこう頼んでください。

### プロンプト例：環境構築

```markdown
あなたはプロのPythonエンジニアです。
Mac（またはWindows）を使っています。

以下の条件で、X（旧Twitter）自動投稿ツールを作るための開発環境をセットアップする手順と、実行すべきコマンドを教えてください。

1. プロジェクト名: antigravity_sns
2. 仮想環境（venv）を使用する
3. 必要なライブラリ: tweepy, python-dotenv
4. フォルダ作成からライブラリインストールまでの全コマンドを提示してください。
```

AIは、あなたのOSに合わせて正確なコマンドを吐き出してくれます。
あなたはそれをコピーして、ターミナルに貼り付けるだけです。

## 2. コマンドを実行する

AIが提示したコマンド（`mkdir` や `python -m venv` など）をターミナルで実行してください。
エラーが出たら？ そのままエラーメッセージをコピーしてAIに投げ返せば、解決策を教えてくれます。

## 3. 必要な道具の確認

成功していれば、フォルダの中に `venv` という名前の「隔離されたPython部屋」ができているはずです。

## 4. 【最重要】`.env` ファイルで鍵を隠す

ここがプロとアマチュアの分かれ道です。
**コードの中に直接API Keyを書いてはいけません。**

```python
# 絶対に駄目な例
access_token = "12345abcde..." # スクショをSNSにあげたら最後、アカウントを乗っ取られます
```

代わりに、`.env`（ドット・エンブ）という隠しファイルを作って、そこに鍵をしまいます。

1. フォルダの中に `.env` という名前のファイルを新規作成します。
2. 第一章でメモした4つの鍵を、以下の形式で書き込みます。スペースは入れないでください。

```text
X_API_KEY=あなたのAPI_Key
X_API_SECRET=あなたのAPI_Key_Secret
X_ACCESS_TOKEN=あなたのAccess_Token
X_ACCESS_TOKEN_SECRET=あなたのAccess_Token_Secret
```

準備は整いました。仕組みは作られ、鍵はセットされました。
